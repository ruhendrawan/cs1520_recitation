# Week 13 — Rust Iterators + Closures

This folder has the starter files for the Week 13 recitation.

This week you will use Rust's iterator API to do traversal of list.

## Files

- `src/main.rs` is your starter for iterator and closure practice.

## Setup

```bash
cd recitations/week13-rust-iter-closures
cargo run
```

The starter should build immediately. Then open `src/main.rs` and solve the five
problems in order.

## Learning goals

By the end of the recitation you should be able to:

1. Use iterator adapters such as `filter()` and `map()`.
2. Use consumers such as `sum()`, `product()`, `fold()`, or `reduce()`.
3. Read and write simple closures like `|n| n % 2 == 0`.
4. Explain the difference between borrowing iteration with `iter()` and consuming
   iteration with `into_iter()`.
